# 详情页参考图模式 - 实现计划

## 需求理解

### 核心功能
增加一个"参考图模式"，用户可以上传已有的详情页作为参考，让LLM分析其构图、排版、文字结构、设计理念，然后结合产品图生成相似的详情页提示词。

### 工作流程

```
┌─────────────────────────────────────────────────────────────┐
│                     用户输入                                 │
├─────────────────────────────────────────────────────────────┤
│  产品图数量: 1~3 (选择生效的产品图片数量)                      │
│  图片1~3: 产品图 (根据数量选择生效)                           │
│  参考图模式: 1=启用, 2=禁用                                   │
│  图片4: 详情页参考图 (模式=1时生效)                           │
└─────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────┐
│                    模式判断                                  │
├─────────────────────────────────────────────────────────────┤
│  IF 参考图模式 == 1 且 图片4已上传:                          │
│     → 进入【参考图分析模式】                                  │
│     → Skill技能无效                                          │
│  ELSE:                                                       │
│     → 进入【正常Skill模式】                                   │
│     → 使用原有技能系统                                        │
└─────────────────────────────────────────────────────────────┘
```

## 详细设计方案

### 1. 新增参数

| 参数名 | 类型 | 范围 | 默认值 | 说明 |
|-------|------|------|-------|------|
| `产品图数量` | INT | 1~3 | 3 | 选择生效的产品图片数量 |
| `参考图模式` | INT | 1~2 | 2 | 1=启用参考图, 2=禁用(正常模式) |

### 2. 图片输入逻辑

**产品图片处理 (图片1~3)**:
```python
product_images = []
if 产品图数量 >= 1 and 图片1 is not None:
    product_images.append(图片1)
if 产品图数量 >= 2 and 图片2 is not None:
    product_images.append(图片2)
if 产品图数量 >= 3 and 图片3 is not None:
    product_images.append(图片3)
```

**参考图片处理 (图片4)**:
```python
reference_image = None
if 参考图模式 == 1 and 图片4 is not None:
    reference_image = 图片4
```

### 3. 两种模式的工作逻辑

#### 模式A: 参考图分析模式 (参考图模式=1 且 有参考图)

**系统提示词 (专用)**:
```markdown
你是一位专业的电商详情页分析师和设计师。

【任务说明】
用户上传了1张详情页参考图和{产品图数量}张产品图。

【分析要求】
请深入分析详情页参考图，提取以下要素：
1. 构图布局：整体页面结构、图片位置、留白处理
2. 排版风格：文字区域分布、标题样式、信息层级
3. 设计理念：视觉风格、配色方案、品牌调性
4. 结构模块：包含哪些功能区块（首屏、参数、细节、场景等）

【输出要求】
结合产品图和参考图的设计风格，生成{生图数量}条AI绘图提示词。
每条提示词必须：
- 体现参考图的设计风格
- 适配当前产品的特点
- 包含构图、排版、文字区域说明
```

**用户提示词模板**:
```markdown
【产品信息】
{product_info}

【参考详情页分析】
请参考上传的详情页图片，分析其设计特点。

【需要生成的图片】
共{生图数量}张，参考原详情页的结构模块。

请生成{生图数量}条独立的AI绘图提示词。
```

#### 模式B: 正常Skill模式 (参考图模式=2 或无参考图)

保持原有逻辑不变：
- 使用 `使用技能` 开关控制是否使用Skill
- 使用 `技能选择` 选择具体技能
- 原有所有功能正常工作

```python
def process(self, ..., 产品图数量: int, 参考图模式: str, ...):
    
    # 1. 处理产品图 (根据数量选择)
    product_images = []
    if 产品图数量 >= 1 and 图片1 is not None:
        product_images.append(tensor_to_pil(图片1))
    if 产品图数量 >= 2 and 图片2 is not None:
        product_images.append(tensor_to_pil(图片2))
    if 产品图数量 >= 3 and 图片3 is not None:
        product_images.append(tensor_to_pil(图片3))
    
    # 2. 判断是否进入参考图模式
    is_reference_mode = (参考图模式.startswith("1") and 图片4 is not None)
    
    if is_reference_mode:
        # ===== 参考图分析模式 =====
        reference_image = tensor_to_pil(图片4)
        
        # 构建专用系统提示词
        system_prompt = self._build_reference_system_prompt(
            language=语言,
            platform=平台,
            batch_count=生图数量
        )
        
        # 构建用户提示词
        user_prompt = self._build_reference_user_prompt(
            product_info=product_info,
            batch_count=生图数量
        )
        
        # 调用LLM (产品图 + 参考图)
        all_images = product_images + [reference_image]
        response = self._call_llm_with_reference(
            llm_config=LLM配置,
            system_prompt=system_prompt,
            user_prompt=user_prompt,
            images=all_images,
            product_count=len(product_images)
        )
        
    else:
        # ===== 正常Skill模式 =====
        # 原有逻辑保持不变
        ...
```

#### 4.3 新增方法

```python
def _build_reference_system_prompt(self, language: str, platform: str, batch_count: int) -> str:
    """构建参考图分析模式的系统提示词"""
    return f"""你是一位专业的电商详情页分析师和设计师。

【任务配置】
- 输出语言: {language}
- 目标平台: {platform}
- 生成数量: {batch_count}张详情页图片

【核心任务】
用户上传了详情页参考图和产品图。
请先深入分析参考图的设计特点，然后结合产品生成新的详情页提示词。

【分析维度】
1. 构图布局：页面结构、图片位置、留白处理
2. 排版风格：文字区域、标题样式、信息层级  
3. 视觉风格：配色方案、光影效果、整体调性
4. 结构模块：首屏、参数、细节、场景等区块

【语言规则】
所有输出必须使用 {language}，严禁混合其他语言。
"""

def _build_reference_user_prompt(self, product_info: str, batch_count: int) -> str:
    """构建参考图分析模式的用户提示词"""
    return f"""【产品信息】
{product_info}

【参考详情页分析要求】
最后一张图片是详情页参考图，请分析：
1. 整体构图和页面布局
2. 文字排版区域分布
3. 设计风格和配色方案
4. 包含的功能模块

【生成要求】
请生成{batch_count}条AI绘图提示词，要求：
- 学习参考图的设计风格
- 适配当前产品的特点
- 每行一条独立提示词
- 包含详细的构图和排版说明
"""

def _call_llm_with_reference(self, llm_config, system_prompt, user_prompt, images, product_count):
    """调用LLM（参考图模式专用）"""
    # 构建消息，说明前N张是产品图，最后一张是参考图
    messages = [
        {"role": "system", "content": system_prompt},

            *[{"type": "image_url", "image_url": {"url": f"data:image/png;base64,{pil_to_base64(img)}", "detail": "high"}} for img in images],
            {"type": "text", "text": f"前{product_count}张是产品图，最后1张是详情页参考图。\n\n{user_prompt}"}
```
